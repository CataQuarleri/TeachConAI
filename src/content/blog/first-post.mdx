# What is ChatGPT and how can bilingual teachers use it?

> _¿Qué es ChatGPT y cómo puede usarlo una maestra bilingüe?_

---

**ChatGPT** is an artificial intelligence model developed by OpenAI that can understand and generate human-like text. For bilingual teachers, it’s a powerful ally that can assist with lesson planning, communication with families, and creating bilingual instructional materials.

---

## 🧠 Use Cases for Bilingual Teachers  
### In English:
- Draft lesson plans for dual-language settings  
- Generate classroom signs or materials in both languages  
- Translate notes for families with home language preferences  
- Create reading comprehension questions or writing prompts  

### En español:
- Redactar planes de clase adaptados a estudiantes bilingües  
- Generar carteles o materiales bilingües para el aula  
- Traducir comunicados para las familias  
- Crear preguntas de comprensión lectora o actividades de escritura  

---

## 🎯 Tips for Prompting

- Be specific: "Generate a bilingual reading activity for 2nd grade using the book *Esperanza Rising*."
- Use language tags: "Create a vocabulary list in English and Spanish."
- Review outputs for accuracy — AI is helpful, but not perfect.
